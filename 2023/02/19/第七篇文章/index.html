19.函数重载的作用是在不同的参数如数组和结构与普通的常量时,使得方法不一样,而导致要重新写方法,但要达到的目的是一致的
void Swap(AnyType& a, AnyType& b) {
AnyType temp;
temp = a;
a = b;
b = temp;
}

template
void Swap(T* a, , T* b,int n) {
T temp;
for (int i = 0; i < n; i++) {
temp = a[i];
a[i] = b[i];
b[i] = temp;
}
}
这是模板与重载的结合使用,使得Swap的功能更丰富
